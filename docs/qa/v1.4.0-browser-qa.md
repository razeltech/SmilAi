Student Dashboard - My Profile
---------------------------------
Status:
❌ Broken

Features tested:
- Clicking the "My Profile" tab from the navigation.

Browser Console Errors:
React error: `<StudentDashboard>` component render error. (Rendered a completely blank page).

Network Errors:
None

Backend Errors:
None

Root Cause (best guess):
The `StudentDashboard` component is trying to access a property (like `profile.name` or similar) that is `undefined` when the Profile tab is rendered, causing a fatal React runtime crash.

Severity:
Critical

Recommended Fix:
Add optional chaining (`?.`) or fallback default values to the profile data rendering inside the `StudentDashboard` component.
