# v1.4.0 Language Platform Foundation

## Overview
This milestone establishes the architectural foundation for the SmilAI Language Platform. It does not implement active translation, but sets up the rigid interfaces and abstractions needed for future capability sprints.

## Key Architectural Decisions
- **ADR-001**: Offline-first principle.
- **ADR-002**: English Core architecture.
- **ADR-003**: Deterministic Learning Engine.
- **ADR-004**: Language Layer Middleware.
- **ADR-005**: Provider Abstraction Pattern.
- **ADR-006**: Deployment Profiles.
- **ADR-007**: Voice Registry mapped via persona/locale.

## Components Implemented
- `BaseProvider` and `RequestContext` interfaces.
- `LanguageService` pipeline orchestration (Detect -> Normalize -> Cache -> Glossary -> Translate).
- Strict separation of Voice Engine from language awareness via `VoiceRegistry`.
